This project demonstrates the usage of 100BASE-T Ethernet Expansion Module with Saturn V3 Spartan 6 FPGA Module. You will need the following hardware to test this project.

Saturn - Spartan 6 FPGA Development Board with DDR SDRAM (http://numato.com/saturn-spartan-6-fpga-development-board-with-ddr-sdram)
IO Breakout Module for Saturn (http://numato.com/io-breakout-module-for-saturn)
100BASE-T Ethernet Expansion Module (http://numato.com/100base-t-ethernet-expansion-module)

This project is created using Xilinx EDK 14.7.
